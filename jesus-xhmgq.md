# 四月十三日
```
使徒行传1章3节
    他受害之后，用许多的凭据，将自己活活的显给使徒看，四十天之久向他们显现，讲说神国的事。

    复活节来临了！只是一年中最可爱的季节，我们在这时纪念胜过死亡复活的主。

    严冬以后，阳春重回，鸟儿开始歌唱，枝头露出绿芽，花朵争艳斗丽，上主造化的奇工借着这一切生物表现出来。它们似从熟睡中苏醒，充满蓬勃生机。我们的心也给唤醒过来，思想复活的主。

    主耶稣复活后在世上的四十天中，活活的把他显现给人看，其中有多少奇妙的事，当他打开圣经，向门徒讲解自己是复活主的时候，门徒的心一定像火焚一般难熬！他的显现十分自然。十分轻而易举；他的脚步无声。在黎明的时候，他向马利亚显现，像往常一样叫她的名字。耶稣在提比哩亚海边向他所亲爱的，曾为他放下捕鱼工作的门徒显现。他为这些经过整夜打鱼、筋疲力竭的门徒预备好了早餐。在安息日下午走向以马忤斯的路上，他与两个忧忧愁愁的门徒同行，听他们说“我们素来所盼望要赎以色列民的主，已经被大祭司长定了死罪，钉在十字架上。” 耶稣用自己的显现为他们解开疑窦，让他们能为所成就的一切做活的见证。赞美主荣耀的复活！

    历代信徒都留下了他们的见证，述主上主亲自与他们同在，和那种有主同在的快乐和欢欣。亚伯拉罕在夜里看守坛上的火，摩西在西乃山上，以赛亚在圣殿里，彼得和雅各在变像山上，还有约翰在拔摩海岛，都有过与主同在的经历。我们也会有与基督的荣耀同在的日子！“你们中间有永生上主。”（约书亚记3章11节）“我常与你们同在直到世界的末了。”

    求主开我们的眼睛，让我们看见主基督从死里复活所成就的丰富、荣美和超越的大能！
```

    在牙买加有位教师，规定学生说谎一次，要打手心七板。有天，一位个子很小的女学生说了谎，给叫到这位老师面前受处罚。看见她们又小又脆弱，那位老师实在不忍打她。可是，为了执行校规，又不能随便算数。于是他举起戒尺打了一下，女孩子立刻痛楚地尖叫，那声音直刺入他心底。他住了手，不能继续执法。但学生犯了规，一定得受罚，怎么办呢？他望了在座的男同学一眼，问道：“哪一位愿意替她受完这次处罚？”话刚说完，矮小的明儿马上站起身来，说：“老师，我愿意。”于是毅然跑到先生桌前。一声不响，忍受余下的六板。
    从这个勇敢的孩子身上让我们看见了一个更伟大的爱心，一个同样无罪替人受罪的爱心。那位教师因这孩子的义举深受感动，含着泪，合上书本，把学生叫拢到书桌前，开始向他们简述那位最有爱心的主基督和他怎样担当我们过犯的故事。

# 四月二十三日
```
诗篇86章1节
    求你侧耳应允我，因我是困苦穷乏的。
    有时候我们会奇怪：为什么我们的祷告不能得到上主即时的垂听？为什么上主给我们的并不是我们所祈求的？下面的故事可以帮助我们了解：
    森林里有三棵大树，同时祷告上主，希望锯下来以后，能用来做他们愿望中的材料。第一棵树祈求可以用来做美丽的宫殿，第二棵树希望造成行走远洋的大舟，第三棵树盼望留在森林里，永远朝着天，向着上主。
    有一天，伐木的人来了，把第一棵树砍了下来，没有用它造宫殿，只把它用来造一个普普通通的马槽。可是一切造物中最俊美的圣婴就在这个马槽里出生。第二棵树只造成了一只小船，放在加利利海里，让一个修长的年轻人立在它的舱面上，向群众讲道：“我来是要让众人得生命，并且得的更丰富。”第三棵树造成了一个十字架，人把这位青年人，也就是世人中从未有过的可爱的人子，钉死在这个木架上。从此以后，这个十字架一直指引人归向上主。
    这三棵树的祷告，都蒙了垂听。
        我无论求什么，
          我都深深的相信，
        主必定愿意给我，
          而且给的更美。
        主啊，我因此愿意，
          把一切要求交托你。
        顺服在你脚前，
          因你的旨意最美。

# 四月二十八日
```
约伯记14章16节

    唯愿你把我藏在阴间，存于隐密处······
         愿你为我定了日期，纪念我。
    全能的上主亲自向他的仆人说话的时刻，一定美妙无比。上主很可能这样说：“比撒列，我需要你为我在今天做这件事。”听见主这样向他说话的人，心里一定十分明白主在他身上的旨意和计划。主要他去做的事，并不像他拣选摩西去解救以色列人那么重要。但是在主自己看来，他要比撒列去做的工作仍是极其重要，影响力也绝不下于摩西所做的。当比撒列开始他的工作以后，因为明白这是主派给他的，他一定抱着无比的信心，奋勇向有。这是上主派给他的工作，与任何人无关，连摩西也没有份。他一定心里充满了喜乐和勇气，继续他们的工作，直到会幕做成，让上主住在其中。（参出埃及记36章）
    历史上有不少人，他们在圣经上所记载的只是一个名字，可是他们却完成了上主在他选定的时刻所要他们继续的工程。路加福音二章所提到的西面，就是这样的一个人。他用一生的时间来等待这一件工作，他等待弥赛亚的到来，深深相信他的等待不会落空。
    科学家谈到高山上的花朵怎样埋藏在雪中好几个月的故事，他们虽被大雪掩盖，仍旧充满活力与盼望；等到太阳一出，几个小时后，雪被溶化，花朵会灿烂地一一开放出来。这也是西面的光景，他像埋藏在雪中的花，用一生来等待，等到主公义的阳光出现，冰消雪融，他心花怒放，欣然接受上主给他光荣的责任；籍着祝福抱在他手中的圣婴，预告救世大功的来临。
